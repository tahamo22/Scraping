# Carrefour Egypt Web Scraper (Vegetables & Fruits)

A Python-based web scraping tool that extracts product information (Names, Prices, and Image URLs) from the Carrefour Egypt website using **Selenium**. This script is designed to handle dynamic content and infinite scrolling.

## üöÄ Features
- **Dynamic Content Handling:** Uses Selenium to interact with JavaScript-rendered content.
- **Auto-Scrolling:** Automatically scrolls to the bottom of the page to load all available products.
- **Data Export:** Saves the scraped data into a clean `carrefour_products.csv` file.
- **Comprehensive Extraction:** Collects product titles, current prices, and product image links.
